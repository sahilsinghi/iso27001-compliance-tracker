# Sample Assessment — Acme Health Pvt Ltd

This folder contains a complete sample assessment for **Acme Health Pvt Ltd**, a fictional 340-person Indian SaaS company in the healthcare sector preparing for first-time ISO 27001:2022 certification.

Acme Health is the same fictional persona used in the companion [DPDP Act 2023 Self-Check](https://github.com/sahilsinghi/dpdp-compliance-tool) tool.

## How to Import

1. Open [ISO 27001 Self-Check](https://iso27001-compliance-tracker.vercel.app)
2. Click **Dashboard** → **Import JSON**
3. Select `acme-health-tracker.json`
4. The full assessment loads instantly — company name, assessor, all 93 control states, evidence references, and notes

## About Acme Health

- **Industry**: Healthcare SaaS (patient management platform for Indian hospitals)
- **Size**: ~340 employees, Bengaluru HQ + remote workforce
- **Cloud**: AWS ap-south-1 (Mumbai) primary, multi-AZ
- **Regulatory exposure**: DPDP Act 2023, IT Act 2000, IRDAI cyber guidelines
- **Certification goal**: First ISO 27001:2022 certification, Stage-1 audit target Q3 2026

Acme Health represents a realistic maturity level for a 3-year-old Indian SaaS company that has invested seriously in its security program but still has a few controls in progress.
